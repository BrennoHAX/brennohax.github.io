O que é DOCTYPE html ?


Serve para dizer ao navegador que o tipo de arquivo é html e é importante e tem que ser lido antes do resto do código.

O que é head ?


O head vai ser uma coisa fixa que vai acontecer uma única vez, ou seja não será possível manipulá-la depois da execução.

O que é body ?


O body é manipulável, ou seja você pode "alterar" ele quantas vezes quiser.

O que é br ?

O br serve para dar espaços entre linhas,exemplo: eu escrevi "Oi" em uma linha e "Tudo bem ?" na segunda linha mais quero que tenha uma linha vazia entre eles eu dou o comando ‹br›.

Símbolos HTML

Os símbolos HTML são usados para pegar acentuações ou partes de texto que não podem ser usadas diretamento no HTML.
